# 마음쉼

마음이 답답하거나 생각을 잠시 비우고 싶을 때 들어오는 작은 창가 웹앱입니다.

## 기능

- 햇살 드는 창문, 책상, 노트, 펜, 화분, 머그컵, 쓰레기통으로 구성한 반응형 공간
- 노트에 생각 작성 및 브라우저 자동 저장
- 작성한 생각을 구겨 쓰레기통에 버리는 애니메이션
- 4단계 호흡 안내
- 마음을 다독이는 한 문장
- Web Audio API로 생성하는 잔잔한 빗소리
- 모바일/PC 반응형 지원

## 실행

별도 설치 없이 `index.html`을 열면 됩니다. GitHub Pages 저장소에서는 커밋 후 웹으로 바로 배포됩니다.

## 에셋과 라이선스

이 프로젝트의 방, 창문, 책상, 노트, 펜, 쓰레기통, 화분 등의 그래픽은 외부 이미지 파일을 복제하지 않고 HTML/CSS로 직접 제작했습니다. 따라서 재배포 시 별도의 그래픽 에셋 라이선스 제약이 없습니다.

- 글꼴: Google Fonts의 **Gowun Dodum**, **Nanum Pen Script** (각 폰트의 오픈 폰트 라이선스 적용)
- 아이콘: 운영체제 기본 이모지 글리프 사용
- 효과음: 외부 음원 파일 없이 브라우저 Web Audio API에서 실시간 생성

무료 스프라이트 파일을 무분별하게 복사하는 대신, 라이선스 충돌과 출처 누락을 막기 위해 직접 제작한 CSS 기반 스프라이트형 오브젝트를 사용했습니다.

## 개인정보

작성한 내용은 서버로 전송되지 않으며 현재 브라우저의 `localStorage`에만 저장됩니다.
